<b>ox</b>search – Eine leistungsstarke, mehrsprachige Suchmaschine auf Basis von ASP.NET Core

Ich freue mich, mein neuestes Projekt vorstellen zu dürfen: oxsearch. Dabei handelt es sich um eine spezialisierte Suchmaschine, die auf die Indexierung und Verarbeitung von Webinhalten in Deutsch, Englisch und Persisch optimiert ist.

Technische Highlights & Funktionalitäten:

1. Robustes Backend: Implementiert mit ASP.NET Core MVC unter Verwendung von SQLite für eine effiziente Datenhaltung.
2. Automatisierter Web-Crawler: Ein eigenständig agierender Crawler, der das Web kontinuierlich nach neuen Inhalten durchsucht und den Index aktuell hält.
3. Intelligente linguistische Analyse: Sprachspezifische Normalisierung zur präzisen Verarbeitung von Sonderzeichen (z. B. deutsche Umlaute und persische Schriftzeichen).
4. Dynamisches Ranking-System: Gewichtung der Suchergebnisse nach Relevanz, Aktualität und Popularität (Besucherzahlen).
5. Modernes UI/UX: Ein responsives, mehrsprachiges Interface (aktuell auf Deutsch lokalisiert), realisiert mit Tailwind CSS und Preline UI.
6. Interaktive Features: Nutzer können aktiv neue URLs zur Indexierung vorschlagen; inklusive integrierter Paginierung und Besucherstatistiken.

Das Projekt ist Open Source. Ich lade euch herzlich ein, den Quellcode auf GitHub zu erkunden. Ich freue mich auf technisches Feedback und konstruktive Anregungen!
